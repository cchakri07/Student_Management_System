# Student Management System (Java)

A simple console-based Student Management System built using Java.  
This project allows users to perform basic CRUD operations (Create, Read, Update, Delete) on student records with file handling for data persistence.

#  Features

- ➕ Add new student details  
- 🔍 View student details by ID  
- ✏️ Update existing student information  
- ❌ Delete student records  
- 💾 Data stored permanently using file handling (`student.txt`)  

# Technologies Used

- Java  
- File Handling (File, FileWriter, Scanner)  
- Collections (ArrayList)  

#  How It Works

- All student data is stored in a file (`student.txt`) in CSV format.
- When the program starts, data is loaded into memory (ArrayList).
- Any changes (add/update/delete) are reflected both in memory and in the file.
